Hi, thanks for purchasing a Gooey Template

Following is a list of each html page you'll find in this template, along with a description of how to use it.

Base
Base.html is a great starting point for your interface. This is a basic template, that shows different column layouts.
Fixed
Fixed.html shows a fixed width page style. You can make any page a fixed width by adding the class "gt-fixed" to the body tag. To edit the width of the page, open gt-fixed.css and edit the width.
Tables
Tables.html contains a list of sample tables you can use for things like articles, comments, users, and more. To use a table, just copy the html code into your page.
Forms
Forms.html contains a list of sample forms you can use for things like adding articles, editing user settings, and more. To use a form, just copy the html code into your page.
Text Styles
Text-Styles.html includes sample text styles for custom styles as well as default styles for most common html elements.
Buttons
Buttons.html includes a list of buttons that come with this template. Each button uses html text, so you can easily change the text to anything you'd like.
Login
Login.html includes a login form with a user name and password field.

You'll also find several css files in the css folder:

gt-styles.css
This file contains all the template styles. If you need to make any changes to these styles, we recommend that you use "override.css"
override.css
This is an empty css file where you can add styles to override those in gt-styles.css. Putting override styles in this css file will help keep your changes more organized and easier to manage. (Note: If you add styles to this file, be sure to include this file in your html pages.)
custom.css
This file provides a great place to enter any custom styles you might need. (Note: If you add styles to this file, be sure to include this file in your html pages.)
reset-fonts.css
This is the global css reset. Do not make any changes to this file. If you need to override any of the styles in this file, do so in override.css
